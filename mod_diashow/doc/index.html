Features of DiaShow4

  • Timed intervall of images
  • Optional preview of image per mouswheel
  • Optional timer setting
  • Optional link & text for each image
  • Optional show images in random order

How to configure

  • width & heigth of the slideshow
    • Use this values for the dimensions of your container for the show.
      The images get this size too.
  • CSS for border of container
    • Configure the css for the border of the div containing your show
  • The heigth of the hint for the text
    • In this, the text is shown you chose for each image.
      Give it at least a heigth so that everything can be read
  • Delay between image change, in milliseconds
    • Set the time for the show
  • Show pictures in random order
    • Define if you want to show your images in the order you defined or randomly
  • Show the image in full size on triggering mousewheel while mouseover?
    • Define if you want to show the real image in full size
  • The xpos and ypos of the full image relative to the container
    • Define the position for the container of the full image relative to the show
  • The height and width of the full image
    • Define the heigth and width for the full image. All images should of course have the same size!
  • The css for the hint to view the full image
    • Define the css for the hint
